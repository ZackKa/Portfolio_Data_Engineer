# 📊 Portfolio – Data Engineering & Intelligence Artificielle
## 👋 À propos

Bienvenue sur mon portfolio.

Je suis Data Engineer, spécialisé dans la conception de pipelines de données, l’industrialisation de systèmes IA et la mise en production de solutions data-driven.

Ce portfolio regroupe l’ensemble de mes projets réalisés entre le projet 2 et le projet 13, illustrant une montée en compétence progressive allant du traitement de données jusqu’à la conception d’architectures IA complexes et cloud-ready.

## 🎯 Objectif de ce portfolio

Ce repository a pour objectif de :

- Montrer mon évolution technique sur des projets concrets
- Illustrer ma capacité à construire des pipelines data de bout en bout
- Démontrer mes compétences en IA (NLP, RAG, LLM)
- Mettre en avant mes compétences en industrialisation et cloud
- Servir de base de présentation pour des opportunités professionnelles
